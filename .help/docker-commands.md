# builda o projeto

## `docker-compose up --build`

# ==========================

# sobe o container (sem travar o terminal)

## `docker-compose up -d`

# ==========================

# derruba o container

## `docker-compose down`

# ==========================

# lista os container ativos

## `docker ps -a`

# ==========================

# remove o container pelo id

## `docker rm` *id*

# ==========================

# lista as imagens

## `docker image ls`

# ==========================

# apaga a imagens pelo id

## `docker image rm` *id*
## `docker rmi` *id*

# ==========================
